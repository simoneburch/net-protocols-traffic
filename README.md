<h1>Observe and Filter Network Traffic using Wireshark</h1>
Wireshark is a tool used to analyze and filter network protocols as they are used within the network. This will be a short demonstration of its use combined with command line tools and network security groups to help illustrate some networking activities.<br />

<h2>Environments and Technologies Used</h2>

- Microsoft Azure (Virtual Machines/Compute)
- Remote Desktop
- Wireshark

<h2>Operating Systems Used </h2>

- Linux Ubuntu
- Windows 10 (21H2)

<h2>High-Level Deployment and Configuration Steps</h2>

- Step 1: Create your Azure Resources and Install Wireshark
- Step 2: Ping your Ubuntu VM from your Windows VM (ICMP traffic)
- Step 3: Observe SSH, DHCP, DNS, and RDP Traffic
